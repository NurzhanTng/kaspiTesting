# Бот для отслеживания терминала Kaspi и обновления ссылки на оплату

**Бот для отслеживания терминала Kaspi и обновления ссылки на оплату** — это инструмент автоматизации, предназначенный для отслеживания наличия терминалов Kaspi с помощью камеры. Когда терминал обнаружен, бот автоматически обновляет ссылку на оплату для пользователя, что позволяет совершать онлайн-платежи, обходя ограничения системы банка Kaspi. Этот бот предоставляет удобное решение для совершения онлайн-платежей без необходимости физического взаимодействия с терминалом.

## Основные особенности
- **Обнаружение терминала**: Использует компьютерное зрение для обнаружения терминалов Kaspi в потоке с камеры.
- **Обновление ссылки на оплату**: Автоматически обновляет ссылку на оплату для пользователя, когда терминал обнаружен.
- **Обход ограничений банка Kaspi**: Позволяет пользователю совершать онлайн-платежи, несмотря на ограничения системы банка Kaspi.
- **Мониторинг в реальном времени**: Постоянно отслеживает поток с камеры и обновляет данные мгновенно.
- **Обработка ошибок**: Обеспечивает стабильную работу даже при изменяющихся сетевых условиях или проблемах с камерой.

## Технологический стек
- **Python библиотеки**:
  - **aiohttp**: Асинхронный HTTP клиент/серверный фреймворк.
  - **opencv-python**: Библиотека OpenCV для выполнения задач компьютерного зрения.
  - **pyzbar**: Чтение штрих-кодов и QR-кодов для идентификации терминалов.
  - **numpy**: Числовые операции для поддержки алгоритмов компьютерного зрения.
  - **aiohappyeyeballs**: Асинхронные операции для обработки ввода с камеры и детекции.
  - **marshmallow**: Сериализация и десериализация данных.
  - **python-dotenv**: Управление переменными окружения.
- **Backend**:
  - **Асинхронная обработка** с использованием **aiohttp** для неблокирующих, реальных операций.

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/NurzhanTng/Kaspi-Tracker-Bot.git
   ```

2. Установите все зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Настройте переменные окружения:
   Создайте файл `.env` и укажите необходимые переменные для работы бота, включая параметры камеры и учетные данные для API Kaspi.

4. Запустите бота:
   ```bash
   python main.py
   ```

5. Бот начнет отслеживать поток с камеры и автоматически обновлять ссылку на оплату, когда терминал Kaspi будет обнаружен.
